# Sample output

These files are **placeholder / sample output** representing a standard Salesforce Sales Cloud and Service Cloud org. They are included only to illustrate the pipeline output format.

They are **not** an export from any real or production environment. Run the pipeline against your own Salesforce org to generate actual output.

|---|---|
| `diagrams/` | Sample Mermaid ER diagrams (`.mmd`) |
| `objects/` | Sample object field CSVs |
